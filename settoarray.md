This code snippet is used to convert from a set to an array.


- Firstly , create a constant and assign a set.
- then, create a contant array .
- use [...set] to convert to an array

```js
let set = new Set([3,56,'a string '])
console.log(set)

let array = [...set]                    
console.log(array)
```
